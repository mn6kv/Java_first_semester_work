    • Назначение и прицип действия пожарных дверей

    Главным назначением противопожарных дверей является способность удерживать пламя, дым, продукты сгорания, высокую температуру в замкнутом пространстве определенное количество времени. При пожаре дверное полотно не деформируется, не утрачивает свои несущие свойства, защищает внешнее пространство от проникновения отравляющих газов, опасных для здоровья.

    Противопожарные двери бывают внутренние (межкомнатные) и наружные, сопровождают людей по пути эвакуации. Их использование рекомендуется для жилых и коммерческих зданий. Для лабораторий, цехов химической промышленности, лакокрасочных заводов, производств, связанных с опасностью возгораний и взрывов, их применение обязательно по правилам пожарной безопасности.

    Принцип действия противопожарной защиты прост. В момент распространения огня по территории, двигаясь от первоначального источника, пламя упирается в преграду, не дающую ему идти дальше. Алюминиевый профиль, благодаря специальной термоизолирующей вставке, не нагреваются с внешней стороны. Конструкция остается безопасной, не вызывает ожогов при прикосновении.

    Виды противопожарных дверей

    Компания Михайловские двери на собственном производстве создает несколько видов дверей для пожарной безопасности помещений. Они не отличаются от обычных, соответствуют общей стилистике здания, внутреннему интерьеру помещений. Большинство — светопрозрачные, производятся из алюминиевого профиля с полным остеклением, при желании, можно заказать глухие без изменения защитных свойств. Дверное полотно обладает высокой степенью противопожарной защиты, дающей 15-90 мин. сопротивления распространению огня. Количество минут зависит от толщины стекла и обозначается маркировкой, присваиваемой по результатам огневых испытаний.

    Виды маркировок с расшифровкой

    Буквы показывают характеристики повреждений от огня

    • Е — утрата целостности полотна или стекла
    • R — потеря несущей способности
    • I — утрата теплоизолирующих свойств, повышение температуры полона выше 140 °С
    • W —превышение теплового потока
    • S — превышение задымленности вне помещения, где расположен очаг возгорания

    Цифры: 15, 30, 60 — обозначают количество времени, через которые наступают эти характеристики

    Установка готовой конструкции

    Мы занимаемся оснащением предприятий противопожарными дверями для создания безопасного пространства при возникновении чрезвычайных ситуаций. Кроме поставки мы проектируем, монтируем, сдаем объект проверяющей организации (инспектору пожарной охраны) и получаем на него необходимую документацию. Монтаж проводят профессиональные бригады с учетом требований пожарной безопасности зданий. После установки специалисты представляют готовый объект пожарному инспектору, получают его положительное заключение. Чертежи проекта хранятся у нас в архивах, при возникновении поломок, входящих или не входящих в нашу гарантию, мы восстановим дверное полотно в первозданном виде, заменив не целиком, а только поврежденные элементы. При идеальном совпадении деталей полотно не потеряет своих свойств, будет продолжать защищать здание от огня еще долгое время.
